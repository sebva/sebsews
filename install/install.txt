- INSTALLATION :
	0. !!! PROTÉGEZ LE REPERTOIRE admin/, SANS QUOI N'IMPORTE QUI PEUT CRÉER/MODIFIER/SUPPRIMER DES PAGES ET NEWS VOIRE CAUSER DES DÉGATS !!!
	1. Ouvrez le site web dans votre navigateur, l'installation devrait débuter. Si ce n'est pas le cas, naviguez vers le répertoire install et suivez les instructions.
	2. Supprimez ou renommez le répertoire install.
	3. Customisez votre site ! La documentation nécessaire au moteur de template se trouve dans le répertoire templates.
		
- REMARQUES IMPORTANTES :
	1. Vous pouvez trouver du support par l'intermédiaire de https://sourceforge.net/projects/sebseasywebsite/support
	2. Veillez à ce que l'interclassement MySQL soit utf8_general_ci .
	3. Le support d'Internet Explorer 6 et inférieur n'est pas assuré.
	4. PHP 5 est requis ! (5.3 pour l'installation automatique)
