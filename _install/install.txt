- INSTALLATION :
	0. !!! PROTÉGEZ LE REPERTOIRE admin/, SANS QUOI N'IMPORTE QUI PEUT CRÉER/MODIFIER/SUPPRIMER DES PAGES ET NEWS !!!
	1. Exécutez le fichier install.sql sur votre base de données (à l'aide de PhpMyAdmin par exemple)
	2. Editez le fichier config.php en fonction de votre configuration
	3. Customisez votre site ! Les fichiers éditables sont style.css et base.php
		Pour base.php, les variables les plus utiles pour la présentation sont $title et $text. D'autres variables moins importantes existent également.
		Pour pouvez ajouter une page uniquement pour son entrée dans le menu à l'aide de la checkbox reservée dans la console d'administration. N'oubliez de créer un fichier portant le nom court dans func/.
		
- REMARQUES IMPORTANTES :
	1. Lors de la création d'une page, veillez à ne pas l'appeler par un nom réservé, sauf si nécessaire. Les noms réservés sont les noms des fichiers situés dans func/.
	2. Veillez à ce que l'interclassement MySQL soit utf8_general_ci .
	3. Le support d'Internet Explorer 6 et inférieur n'est pas assuré.
